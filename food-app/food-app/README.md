# Food Application

This is a food application that allows users to search for restaurants and view their details. The application consists of three main components: Header, Body, and Footer.

## Project Structure

```
food-app
├── src
│   ├── components
│   │   ├── Header
│   │   │   ├── Header.js
│   │   │   └── Header.css
│   │   ├── Body
│   │   │   ├── Body.js
│   │   │   ├── SearchBar.js
│   │   │   ├── RestaurantList.js
│   │   │   └── Body.css
│   │   └── Footer
│   │       ├── Footer.js
│   │       └── Footer.css
│   ├── App.js
│   ├── App.css
│   ├── index.js
│   └── index.css
├── package.json
└── README.md
```

## Features

- **Header**: Contains navigation links for Home, Cart, and About.
- **Body**: Includes a search bar for searching restaurants and a list to display the restaurants.
- **Footer**: Displays footer information.

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. Run `npm install` to install the dependencies.
4. Run `npm start` to start the application.

## Usage

- Use the search bar to find restaurants by name or type.
- Click on the navigation links in the header to navigate through the application.
